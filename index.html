Basic HTML Cheat Sheet

HTML Basics

Learn the building blocks of web pages

Basic cheat sheet. Use the search bar to find a section, copy any code block.

HTML Document Structure

This is the basic skeleton of every HTML page. It tells the browser where the page starts, where the content lives, and what title appears in the tab. 

<!DOCTYPE html>
                <html lang="en">
                <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>My Web Page</title>
                </head>
                <body>
                <h1>Hello, world!</h1>
</body>
</html>

Headings

Headings organize content into levels. h1 is the most important, and h6 is the least important. 

<h1>Main Title</h1>
                        <h2>Section Title</h2>
                        <h3>Subsection Title</h3>
                        <h4>Smaller Heading</h4>
                        <h5>Even Smaller Heading</h5>
                        <h6>Smallest Heading</h6>

Paragraphs

Use paragraphs to write normal text. They are one of the most common tags in HTML and automatically add spacing around the text. 

<p>This is a paragraph.</p>
              <p>This is another paragraph.</p>

Links

Links connect one page to another. The href attribute stores the destination URL. 

<a href="https://example.com" target="_blank">
	Visit Example
</a>

Images

Images are added with the img tag. Always include descriptive alt text for accessibility. 

<img src="cat.jpg" alt="A cat sitting on a couch" width="400">

Lists

Use unordered lists for bullets and ordered lists for numbered steps. Each item goes inside an li tag. 

<ul>
	<li>Apples</li>
	<li>Bananas</li>
	<li>Oranges</li>
</ul>

<ol>
	<li>Wake up</li>
	<li>Learn HTML</li>
	<li>Build a page</li>
</ol>

Tables

Tables are useful for showing structured data in rows and columns. Use th for headers and td for data cells. 

<table>
	<tr>
		<th>Name</th>
		<th>Age</th>
	</tr>
	<tr>
		<td>Ava</td>
		<td>12</td>
	</tr>
</table>

Forms

Forms collect information from users, such as names, email addresses, or messages. Inputs and labels are the building blocks. 

<form>
	<label for="name">Name:</label>
	<input type="text" id="name" name="name">

	<label for="message">Message:</label>
	<textarea id="message" name="message"></textarea>

	<button type="submit">Send</button>
</form>

Buttons

Buttons are clickable elements that trigger actions like submitting a form or running JavaScript. 

<button type="button">Click Me</button>
<button type="submit">Submit Form</button>

Input Types

HTML provides many input types for different kinds of data. Choosing the right type improves user experience on mobile and desktop. 

<input type="text" placeholder="Your name">
<input type="email" placeholder="you@example.com">
<input type="password" placeholder="Password">
<input type="number" min="1" max="10">
<input type="date">
<input type="checkbox"> I agree

Div and Span

div is a block-level container used to group larger sections, while span is inline and good for small pieces of text. 

<div class="card">
	<h3>Profile</h3>
	<p>Hello, my name is <span>Mia</span>.</p>
</div>

Comments

Comments help you leave notes in your code. Browsers ignore them, so they do not appear on the page. 

<!-- This is a comment -->
<p>This text appears on the page.</p>

Semantic HTML Tags

Semantic tags describe the meaning of content, which helps with accessibility, SEO, and code readability. 

<header>Site Header</header>
<nav>Navigation Links</nav>
<main>
	<section>
		<article>Article content</article>
	</section>
</main>
<footer>Site Footer</footer>

Audio and Video

Use the audio and video tags to embed media. Add controls so the user can play, pause, and adjust volume. 

<audio controls>
	<source src="song.mp3" type="audio/mpeg">
	Your browser does not support audio.
</audio>

<video controls width="320">
	<source src="movie.mp4" type="video/mp4">
	Your browser does not support video.
</video>

Basic Meta Tags

Meta tags live inside the head section and give the browser useful information about the page. 

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="A beginner-friendly HTML cheat sheet">
<meta name="author" content="Your Name">
